# Portfolio Images

Add your portfolio images to the appropriate folders:

- **wedding/** - Bridal and wedding editorial photos
- **saree-branding/** - Traditional saree campaign photos
- **makeup/** - Beauty and makeup artistry photos
- **aesthetic/** - Artistic and creative photography

## Supported Formats
- JPG/JPEG
- PNG
- WebP (recommended for web performance)

## Best Practices
- Use descriptive filenames (e.g., `bridal-portrait-1.jpg`)
- Optimize images before uploading (recommended max 2MB per image)
- Maintain consistent quality across all images
- Consider using landscape, portrait, and square orientations for variety

## Adding Images to Your Portfolio

1. Add your images to the appropriate folder
2. Update `src/data/portfolioContent.ts` with the image paths
3. Import images at the top of the file if needed
4. Add image metadata (alt text, aspect ratio) in the gallery arrays
